# User Guide

## Getting Started

ask.app is a full-stack arbitrage platform that allows you to monitor blockchain activity, manage your account, and leverage AI-assisted code generation.

## Account

### Logging In

1. Navigate to the application URL.
2. Click **Log In** in the navigation bar.
3. Enter your email and password.
4. On first login you will be prompted to change your password.

### Dashboard

After logging in you will see the **User Dashboard**, which includes:

- **Account Overview**: Your profile, role, and account status.
- **Activity Metrics**: Recent API calls, blockchain queries, and usage history.
- **Notifications**: System alerts and account notifications.
- **Account Settings**: Update your password, email, and preferences.

### Metered Usage

The **Usage Panel** in your dashboard shows:
- Current period API calls
- Remaining quota
- Historical usage charts

## Navigation

The top navigation bar provides tabs for:

| Tab | Description |
|---|---|
| Home | Platform overview and quick stats |
| Dashboard | Your personal dashboard |
| Users | User directory (Admin/Developer only) |
| Admin | Administration panel (Admin only) |
| Developer | Developer tools (Developer/Admin only) |
| Settings | Account and app settings |
| Docs | This documentation |

## API Access

Users with **Developer** or **Admin** roles can access API keys from the **Developer** tab:

1. Navigate to **Developer → API Keys**.
2. Click **Generate New Key**.
3. Copy the key — it will only be shown once.
